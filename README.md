# MSCloudMusic
在线音乐播放器，提供搜索在线音乐资源，在线播放，下载歌曲，本地音乐播放，本地音乐扫描，推荐MV播放等<br>

项目名称
======
MSCloudMusic是一个在线音乐播放器，听搜索在线音乐资源，在线播放，下载歌曲，本地音乐播放，本地音乐扫描，推荐MV播放等<br>
里面涉及了四大组件中的三个（除了ProviderContent）、多文件下载、断点续传、滑动冲突问题、自定义View等，可以作为一个练手的小项目再适合不过！！！
